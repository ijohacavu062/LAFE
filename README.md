## Abstract
This repository contains the MATLAB implementation of the proposed **LAFE: A Lightweight Edge-aware Nonlinear Fusion Framework for Nighttime HDR Image Enhancement** method described in the submitted paper.

## Requirements
*   **Software:** MATLAB (Tested on R2022a)
*   **Toolboxes:** Image Processing Toolbox

## How to Run
1.  Clone or download this repository.
2.  Open MATLAB and add the root folder to the path.
3.  Run the script `LAFE_method.m`  to reproduce the results.
